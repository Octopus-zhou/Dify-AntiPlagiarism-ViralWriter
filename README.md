# Dify-AntiPlagiarism-ViralWriter
# 🚀 基于 Multi-Agent 阶梯对抗与容灾自愈的爆款文案重写系统

这是一个基于 **Dify 工作流** 与 **自研 Python 查重网关** 编排的工业级全自动内容生产系统。它不仅能像素级重写自媒体爆款，更能死守平台查重红线，具备高可用容灾能力。

## ✨ 核心技术方案
1. **逆向工程先验**：拒绝盲目重写，首创将原爆款解构为 JSON 骨架再行渲染。
2. **物理循环展开**：顺应 DAG 规范，物理铺开 3 轮对抗，且每一轮的 Prompt 惩罚烈度阶梯式递增。
3. **双轨容灾策略**：
   - **主路**：无缝降级为冷封保底 Agent（Temp=0.1），保障高可用。
   - **旁路**：异步触发 HTTP POST 请求，推送实时告警至飞书/企微机器人。

## 📦 如何使用？
1. 下载本仓库的 `文本相似度查重算法与分发数据字典`。
2. 前往 Dify 工作流面板，点击「导入 DSL 」。
3. 绑定你自己的大模型 Key 并在 Python 节点中配置好变量。
4. 点击「测试运行」，输入一段爆款文本，即可见证全自动流转。
